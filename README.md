# CryptoPulseAI

A Flask web app for cryptocurrency price predictions and market analysis.

## Features
- **Prices**: Real-time prices for Bitcoin, Ethereum, Solana, Cardano, Dogecoin.
- **Predictions**: Advanced models for Bitcoin, Ethereum, Solana, Cardano.
- **Charts**: RSI, volume, price trends with Chart.js.
- **UI**: Sleek, dark-themed, responsive design.
